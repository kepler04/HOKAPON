# Imágenes del Hero (banners rotativos)

Coloca aquí tus 3 imágenes con EXACTAMENTE estos nombres:

- `banner-1.jpg`
- `banner-2.jpg`
- `banner-3.jpg`

## Recomendaciones
- **Proporción:** 16:7 horizontal (ej. 1600 × 700 px o más grande).
- **Formato:** `.jpg`, `.png`, `.webp` o `.avif`. Si usas otro formato distinto a
  `.jpg`, cambia la extensión en `src/app/(public)/page.tsx` (constante `HERO_SLIDES`).
- **Texto legible:** el sitio pone un degradado oscuro de izquierda a derecha
  encima de la imagen para que el título blanco se lea. Deja la parte IZQUIERDA
  de tu imagen más despejada (el texto va a la izquierda).
- **Peso:** intenta que cada una pese < 500 KB para que cargue rápido.

Una vez que pongas los 3 archivos aquí, recarga la página: aparecerán
automáticamente en el carrusel (reemplazando los degradados de relleno).
